
# NN Toolkit

A set of helpers to interact with files generated by NN.

NN (Nationale-Nederlanden) is one of the largest insurance and asset management companies in the Netherlands.

- https://en.wikipedia.org/wiki/NN_Group

This toolkit includes support to parse PDF files containing Contributions Statement for the Romanian Mandatory Private Pension. These files can be exported from NN Direct mobile application.

## NuGet Package

```
Install-Package DustInTheWind.NN.Toolkit
```

Or via the .NET CLI:

```
dotnet add package DustInTheWind.NN.Toolkit
```

## Contributions Files

### Step 1 - Export PDF Contributions File

- Open NN Direct mobile application

- Login
- Open "Pensie privata obligatorie"
- Tap "Istoric contribuții"
- Select entire time interval
- Tap three dots -> "Descarcă raport"

This will download a PDF document containing all contributions to the the Romanian Mandatory Private Pension.
